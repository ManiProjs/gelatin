## ✨ Features

- 🎬 Browse your Jellyfin libraries
- 📺 Watch movies and TV shows
- 🎵 Listen to your music collection
- 🔍 Search across your media
- ▶️ Resume playback
- 📜 Continue Watching section
- 🌙 Material 3 interface with dark mode
- ⚡ Fast and lightweight
- 🔒 Secure Jellyfin authentication
- 🖼️ High-quality artwork and metadata

More features are planned!

# Building

## Requirements

Before building, install:

- Flutter SDK
- Dart SDK (included with Flutter)

Verify your installation:

```sh
flutter doctor
```

## Clone

```sh
git clone https://github.com/ManiProjs/gelatin.git
cd Gelatin
```

## Install dependencies

```sh
flutter pub get
```

## Run (Debug)

```sh
flutter run
```

## Build

Build for your desired platform.

### Android

```sh
flutter build apk
```

### Linux

```sh
flutter build linux
```

### macOS

```sh
flutter build macos
```

### Windows

```sh
flutter build windows
```

### Web

```sh
flutter build web
```
